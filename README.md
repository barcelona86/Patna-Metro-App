## 📱 About Patna-Metro-App

The Patna-Metro-App is a modern mobile app made to help you travel the Patna metro network with ease. It uses smart route finding to offer the best routes for your trips. The app supports two languages, Hindi and English, which makes it easy for more people to use. It also features an AI voice assistant named Bodhi. Bodhi can answer questions and help you find your way without typing.

This app was built with popular tools like React Native for the mobile part and Spring Boot for the backend server. Both tools work behind the scenes, so you get a smooth app experience on your phone.

## 🛠 Troubleshooting Tips

- If the app does not install in the emulator, make sure you have allowed installations from unknown sources in the Android emulator settings.

- If the voice assistant does not respond, check that your computer’s microphone is enabled and allowed for the emulator.

- For slow performance, try closing other programs to free up memory.

- Make sure your emulator and Patna-Metro-App are both updated to the latest versions.

- If the app crashes, restart the emulator and try again.
